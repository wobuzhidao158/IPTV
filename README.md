### 📁 项目文件说明
- `update.py`：核心脚本，负责源采集、测速、排序、M3U生成
- `.github/workflows/auto-update.yml`：GitHub Actions自动化配置，每日自动运行
- `iptv.m3u`：最终直播源文件，播放器直接导入
- `update_log.txt`：每日更新日志，记录有效源数量、更新时间

---

### 📱 我的使用指南
1.  **获取直链**：复制 `https://raw.githubusercontent.com/你的GitHub用户名/IPTV_Personal/main/iptv.m3u`
2.  **导入播放器**：打开电视/手机直播软件，导入该直链
3.  **直接观看**：打开即看最新直播源，央视/卫视顺序规整，活源满满
4.  **无需维护**：每日自动更新，永远不用手动折腾，躺平使用
